# sandox-vr-assignment

## Project Scope
- Task 1, 2, 3
- Leaderboard

## Prerequisites

| Node Version | ~20.19.4 |
|---|---|

## Setup

1. Install NPM packages
    ```
    npm i
    ```
1. Build (root directory)
    ```
    npm run build:all
    ```
1. Serve
    ```
    npm run serve
    ```
1. Reach `http://localhost:3000` to mark your ranking!
